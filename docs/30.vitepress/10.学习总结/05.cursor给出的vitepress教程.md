### 一、这个项目大致是什么

- 项目性质：这是一个基于 VitePress + `vitepress-theme-teek` 主题 搭建的文档 / 博客 / 知识库网站模板，已经被你改成了自己用的站点，站点标题是 `LiuMang`。
- 技术栈：
  - 文档框架：`vitepress`
  - 主题：`vitepress-theme-teek`（一个扩展默认主题、支持博客风 + 文档风的主题）
  - 前端：`vue@3`
  - 插件/扩展：时间线插件、自动 frontmatter 插件、代码组图标插件、统计分析、评论系统等。
- 运行方式（看 `package.json`）：
  - 本地开发：`pnpm docs:dev`（或 `npm/pnpm/yarn` 对应命令，模板推荐用 `pnpm`）
  - 构建：`pnpm docs:build`
  - 预览：`pnpm docs:preview`

项目的主要内容都在 `docs` 目录下，`docs/.vitepress` 里是站点配置和主题扩展，`docs/**/*.md` 是你的实际文章内容（比如 `10.运维/10.linux/10.vim.md` 这类）。

------

### 二、目录与功能大致情况

#### 1. 文档内容结构

从 `docs` 下的结构来看，你已经把自己的知识按主题分好了类，比如：

- `10.运维`：Linux、数据库（MySQL、Oracle）、运维总览等
- `15.前端`：HTML、CSS 等
- `20.编程`：Python、Go 等
- `25.黑客`：安全相关
- `30.专题`：脚本、Git、博客搭建等
- `35.工具`：各种工具、配色、画图软件等
- `40.生活`、`45.精神小屋`、`50.娱乐`、`55.兴趣`：偏生活/兴趣向内容
- `60.关于`：关于我、关于本站、友链、时间轴等
- `65.Teek`：关于 Teek 主题本身的文档

同时还有一些 `@pages` 特殊页面，例如：

- `@pages/tagsPage.md`、`categoriesPage.md`、`archivesPage.md`、`articleOverviewPage.md`
  这些是主题用来生成标签页、分类页、归档页、文章索引页的。

整体来说，这是一个 “个人知识库 + 博客 + 生活相册/文案小站” 的综合站点。

#### 2. 站点 & 主题配置（`docs/.vitepress/config.ts`）

主要特性：

- 主题基础配置（`teekConfig`）
  - 壁纸模式：`wallpaper.enabled: true`，首页顶部可以按 F11 全屏壁纸。
  - 博主信息卡片（侧边栏）：
    - 昵称：`Liumang`
    - 座右铭：`明心静性，爱自己`
    - 头像、本地图片、背景图等都已经配好。
  - 分类卡片 / 标签卡片 / 精选文章卡片：
    - `category`、`tag`、`topArticle` 都是启用状态，可分页、自动翻页配置已写好。
  - 文章分析与统计：
    - 字数统计、阅读时长：`wordCount: true, readingTime: true`
    - 访问统计：`docAnalysis` 使用 不蒜子（busuanzi） 的 JSONP 接口，带重试逻辑。
  - 风险链接提示页：
    - `riskLink.enabled: true`
    - 白名单中包含作者原站相关域名（`onedayxyy.cn` 系列、`vp.teek.top` 等）以及 GitHub/Gitee 的正则。
  - 交互增强：
    - 回到顶部按钮：`backTop.enabled: true`，并弹出 `"返回顶部成功"` 提示。
    - 滚动到评论区：`toComment.enabled: true`，提示 `"已抵达评论区"`。
  - 代码块增强：
    - 大段代码自动折叠、底部遮罩、复制成功提示 `"复制成功！"`。
  - 自动封面图：
    - `autoFrontmatter` 插件启用，如果文章没有 `coverImg`，会从 `CoverImgList` 随机生成一个封面图。
- 文章相关功能
  - 旧文提示：
    - `articleTopTip`：如果文章 `date` 超过半年，会在顶部提示“文章发布较早，内容可能过时，阅读注意甄别。”
  - 评论：
    - 使用 `twikoo`：`envId: "https://twikoo.onedayxyy.cn/"`，版本 `1.6.41`。
  - 打赏/赞赏：
    - `appreciation` 配置在文章下方，使用微信支付图标和两张本地二维码图片（`/img/alipay/1.png`、`2.png`）。
  - 单文章 Banner：
    - `articleBanner.enabled: true`，可以显示分类、标签、封面图。
- 网站级配置（`defineConfig` 部分）
  - `title: "LiuMang"`，`lang: "zh-CN"`，`cleanUrls: true`（链接无 `.html` 后缀）。
  - `sitemap.hostname: "https://onedayxyy.cn"`（目前还是模板作者的域名）。
  - `themeConfig`：
    - Logo：`/favicon.ico`
    - 中文 UI 文案：返回顶部、导航标题等。
    - 导航栏：使用 `Nav` 配置（见下）。
    - `socialLinks`: 使用 `SocialLinks` 配置。
    - 搜索：Algolia，配置了一组 appId / apiKey / indexName（`onedayxyy`）。

#### 4. 主题扩展（`docs/.vitepress/theme/index.ts`）

- 继承了 `vitepress-theme-teek`：`extends: Teek`
- 额外引入了：
  - 各种 Teek 提供的样式（导航、侧边栏、代码块、banner 等）。
  - 自定义样式：`./styles/code-bg.scss`、`./styles/iframe.scss`、`./style/index.scss`。
  - 时间线样式：`vitepress-markdown-timeline/dist/theme/index.css`
  - 页面切换进度条：`nprogress-v2`，在路由切换时显示顶部加载条。
  - 全局友链组件 `friend-link`（`SLink`）、情侣相册组件 `CoupleAlbum` / `PhotoCard`。
  - 复制事件 `useCopyEvent`：在浏览器环境下监听复制，复制时弹出提示。
- 最终返回一个自定义的 `Layout` 组件，用来包裹主题的默认布局。
