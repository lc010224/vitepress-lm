# netcup-monitor安装

## 创建文件夹

```
mkdir -p /home/netcup-monitor/data
cd /home/netcup-monitor
```
## 新建compose文件
```
nano docker-compose.yml
```
## compose内容
```
version: '3.8'
 services:
  netcup-monitor:
    image: ghcr.io/agonie0v0/netcup-monitor:latest
    container_name: netcup-monitor
    restart: unless-stopped
    # 端口映射 (宿主机端口:容器端口)
    ports:
      - "5000:5000"
    volumes:
      # 数据与配置文件持久化
      - /home/netcup-monitor/data:/app/data
      # 宿主机时间同步
      - /etc/localtime:/etc/localtime:ro
      # (可选) 若需自动重启 Vertex 容器，必须挂载此 Socket
      - /var/run/docker.sock:/var/run/docker.sock
    environment:
      - TZ=Asia/Shanghai
```

## 创建

```
docker compose up -d
```
