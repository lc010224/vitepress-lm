# **fail2ban安装**

## 安装工具

> 说明：

```
sudo apt update && sudo apt install fail2ban -y
```
## 创建本地配置文件
```
cat <<EOF | sudo tee /etc/fail2ban/jail.local
 [sshd]
 enabled = true
 port = ssh
 filter = sshd
 logpath = /var/log/auth.log
 maxretry = 3
 findtime = 600
 bantime = 3600
 EOF
```
## 重启服务
```
sudo systemctl restart fail2ban
```


>以上代码的意思是10分钟内密码输错3次拉黑IP一小时，
