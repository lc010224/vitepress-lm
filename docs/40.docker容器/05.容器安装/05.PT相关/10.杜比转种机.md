# 杜比转种机安装

## 创建文件夹

```
mkdir -p /home/dolbyreseeder/data
cd /home/dolbyreseeder
```

## 打开compose

```
nano docker-compose.yml
```

## compose文件

```
services:
  app:
    image: hddolby/dolbyreseeder:latest
    environment:
      - NODE_ENV=production
      - DB_FILE=data/dolby.sqlite
      - JWT_SECRET=0102ilyyy..
    restart: unless-stopped
    ports:
      - '3055:3000'
    volumes:
      - /home/dolbyreseeder/data:/app/backend/data
```

## 运行compose

```
docker compose up -d
```
