# docker安装vt和qb

## 一、安装vt

### 拉取容器

```
docker pull lswl/vertex-base:latest
```
### 安装

```
docker run -d
  --name vertex
  -v /home/vertex:/vertex
  -p 3002:3000
  -e TZ=Asia/Shanghai
  --restart unless-stopped
  lswl/vertex:stable
```
 >日志中查看密码

## 二、安装qb

```
docker run -d \
 --name==qbittorrent9 \
 -e PUID==1000 \
 -e PGID==1000 \
 -e TZ==Asia/Shanghai \
 -e WEBUI__PORT==9090 \
 -p 9090:9090 \
 -p 6889:6889 \
 -p 6889:6889/udp \
 -v /home/ilyyy/qbittorrent/config:/config \
 -v /home/ilyyy/qbittorrent/Downloads:/downloads \
 --restart unless-stopped \
 linuxserver/qbittorrent:5.0.4
```
>日志中查看密码
