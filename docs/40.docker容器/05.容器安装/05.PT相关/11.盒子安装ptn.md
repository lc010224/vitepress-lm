# **盒子安装ptn**

## 创建文件夹

```
mkdir -p /home/pt-nexus/data
cd /home/pt-nexus
```
## 新建compose文件
```
nano docker-compose.yml
```
## compose内容
```
services:
  pt-nexus:
    image: ghcr.nju.edu.cn/sqing33/pt-nexus:latest
    container_name: pt-nexus
    ports:
      - 5274:5274
    volumes:
      - /home/pt-nexus/data:/app/data
      - /home/admin/qbittorrent/Downloads:/pt # 视频文件存放路径
      # 如要使用转种上盒功能，tr需要种子文件存放路径，qb使用api无需设置
    environment:
      - TZ=Asia/Shanghai
      # - http_proxy=http://192.168.1.100:7890 # 代理服务器
      # - https_proxy=http://192.168.1.100:7890 # 代理服务器
      - UPDATE_SOURCE=gitee # 更新源，可选: gitee 或 github，不设置默认gitee
      - DB_TYPE=mysql
      - MYSQL_HOST=1Panel-mysql-mZYN
      - MYSQL_PORT=3306
      - MYSQL_DATABASE=pt_nexus
      - MYSQL_USER=ptn
      - MYSQL_PASSWORD=0102ilyyy..
```

## 创建
### /home/pt-nexus路径下运行
```
docker compose up -d
```
