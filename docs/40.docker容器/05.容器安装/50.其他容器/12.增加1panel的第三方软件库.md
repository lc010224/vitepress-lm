# **增加1panel的第三方软件库**

## 运行命令

```
git clone -b localApps https://github.com/okxlin/appstore /home/1panel/resource/apps/local/appstore-localApps
```
```
cp -rf /home/1panel/resource/apps/local/appstore-localApps/apps/* /home/1panel/resource/apps/local/
```
```
rm -rf /home/1panel/resource/apps/local/appstore-localApps
```

然后应用商店刷新本地应用即可

>注意：/home路径根据1panel的安装路径确定，默认是/opt
