# vaultwarden密码管理器安装

## 创建文件夹

```
mkdir -p /home/vaultwarden/data
cd /home/vaultwarden
```
## 新建compose文件
```
nano docker-compose.yml
```
## compose内容
```
version: '3'

services:
  vaultwarden:
    image: vaultwarden/server:latest
    container_name: vaultwarden
    restart: unless-stopped
    volumes:
      - /home/vaultwarden/data:/data  # 数据存储位置
    environment:
      - WEBSOCKET_ENABLED=true  # 开启 WebSocket 支持（手机端同步更快）
      - SIGNUPS_ALLOWED=true    # 首次安装设为 true，注册完账号后一定要改成 false
    ports:
      - "3080:80"     # 映射到本地 8080，让 Caddy 反代，不直接暴露
      - "3012:3012"   # WebSocket 端口
```

## 创建

```
docker compose up -d
```
